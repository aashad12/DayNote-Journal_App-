
DayNote Personal Journaling Desktop Application.

DayNote is a safe offline desktop program meeting the goal of re-inventing personal journaling.
It enables users to make daily journal entries, mood, emotional pattern and privacy protection by use of PIN-based security. All the information is locally stored in SQLite that guarantees privacy and offline availability.

Overview

Conventional journaling is not structured, searchable or emotional. DayNote is a solution to these shortcomings, including a powerful digital journaling app that allows analytics, streak tracking, and secure local storage.

Key Features

Journal Entry Management

* Add, modify, access, and delete journal entries.
* A journal entry on a daily basis to facilitate tracking of streaks.
* Content with rich-text or Markdown support.
* System generated dates of creation and date of update.

 Mood Tracking

* One needed main mood with each entry.
* Two optional secondary moods.
* Positive, Neutral and Negative moods.
* Analytics, emotional trend analysis.

 Tags and Categories

* Support of predefined and custom tags.
* Organizing entries in categories.

Navigation and Search

* Date navigation by calendar date access.
* Page based timeline that allows the viewing of entries.
* Search by title or content moods, and tags filters.

 Streak Tracking

* Current journaling streak
* Longest streak achieved
* Tracking of missed days in the past thirty days.

 Analytics and Insights

* Mood distribution analysis
* Most frequent moods
* Most used tags
* Tag usage breakdown
* Word count trends over time
* Date filterable analytics.

Security and Privacy

* PIN-based application lock
* The secure PIN storage should be implemented with the hash and salt (PBKDF2).
* On-site data storage and independent of clouds.

Export Functionality

* Export free journal file as PDF.
* AA Export selection by date range.
* Printable clean layout of documents.

Theme Customization

* Light and dark theme support
* Theme preference locally saved.


Technology Stack

* Programming Language: C#
* Framework: .NET (Desktop)
* Database: SQLite
* PDF Generation: QuestPDF
* Security: PBKDF2 (Rfc2898DeriveBytes)
* Service based design: Architecture.
